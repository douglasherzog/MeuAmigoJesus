# Meu Amigo Jesus ✝️

Jogo interativo e lúdico para crianças aprenderem sobre a vida de Jesus Cristo, do nascimento à ressurreição.

## Como funciona

A Yoyo acompanha Jesus em 13 fases interativas:

1. ⭐ Nascimento em Belém
2. 💧 Batismo no rio Jordão
3. 🪨 Tentação no deserto
4. 🎣 Pescadores de homens
5. 😊 Bem-aventuranças no monte
6. 👁️ Cura do cego
7. 🍞 Multiplicação dos pães e peixes
8. 🌊 Jesus anda sobre as águas
9. 🌳 Zaqueu sobe na árvore
10. 🌿 Entrada em Jerusalém
11. 🍞 A Última Ceia
12. ✝️ O amor de Jesus
13. 🌅 A Ressurreição

## Características

- **TTS (Text-to-Speech)**: Todas as interações falam em português
- **Visual e intuitivo**: Emojis grandes, cores vibrantes, fontes grandes
- **Animações**: Cada fase tem animações únicas e interativas
- **Sistema de medalhas**: 13 medalhas para conquistar
- **Sistema de estrelas**: Acumule estrelas a cada fase
- **Progresso salvo**: Continua de onde parou

## Como rodar

Abra `index.html` no navegador, ou inicie um servidor local:

```bash
python -m http.server 8080
```

Acesse `http://localhost:8080`

## Tecnologias

- HTML5
- CSS3 (animações, gradientes, responsivo)
- JavaScript puro (sem dependências)
- Web Speech API + servidor TTS local (opcional)
