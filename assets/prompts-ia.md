# Prompts de IA para Ilustrações - Meu Amigo Jesus

Use estes prompts no **Midjourney**, **DALL·E 3**, **Leonardo.ai** ou **Bing Image Creator**.

## Estilo visual recomendado

Adicione no final de cada prompt:

```
... cute children's book illustration, soft warm pastel colors, gentle lighting, rounded friendly shapes, storybook art style, high quality, no text, no watermark
```

Para **fundo**: pedir `wide scene background, no characters, no transparent background, full color illustration`

Para **personagens/objetos**: pedir `transparent background, PNG cutout, single isolated character, soft shadows`

> Nota: para fundos, prefira formatos **WebP/JPG** em 1920x1080. Para personagens e objetos, prefira **PNG transparente** em 512x512 a 1024x1024.

## Dicas de geração

1. **Gere várias versões** de cada personagem e escolha a mais simpática.
2. **Mantenha consistência de estilo**: use o mesmo estilo em todas as fases.
3. **Personagens bíblicos**: peça para não serem muito realistas/photográficos — mantenha ilustração infantil.
4. **Fundo não transparente** para backgrounds.
5. **Personagens transparentes** para melhor posicionamento no jogo.
6. **Ajuste posições** depois no `phases.js` usando `bottom`, `left`, `top`, `right`, `width`.
